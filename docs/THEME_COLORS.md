### Material Palettes Used

All themes use Material Design 3 palettes:
- **Primary**: Violet Palette (`mat.$violet-palette`)
- **Tertiary**: Rose/Cyan Palette

You can change these in each theme file by modifying the `mat.define-theme()` configuration.

### Available Material Palettes

- `mat.$red-palette`
- `mat.$pink-palette`
- `mat.$purple-palette`
